## 주의사항

- 로또는 사행성 소재 — 공개 배포 시 "당첨 확률을 높인다"는 식의 허위·과장 표현을 절대 넣지 않는다.
- "수학적으로 많이 나올법한 번호", 빈도가중 생성은 확률적 근거가 없는 **재미 요소**임을 UI에 명시한다.
- 연금복권은 로또 6/45와 번호 형식이 완전히 달라 별도 데이터·로직이 필요하다 (규칙 확인 필요).
- 당첨번호 데이터 출처의 이용약관·저작권을 확인 전까지 사실 근거로 쓰지 않는다.
