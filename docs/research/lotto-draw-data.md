# 로또 6/45 당첨번호 데이터 출처·이용약관 조사

- 조사일: 2026-07-06
- 조사 방법: 사용자가 외부 도구로 직접 조사, 출처·결론 확인 후 기록 (development-workflow "외부 조사·자문" 절)
- 관련 문서: docs/design/lotto-mvp.md 의 "실데이터 출처·이용약관 미검증" 배포 게이트, src/data/README.md

## 질문

① 로또 6/45 역대 당첨번호(회차·추첨일·본번호 6개·보너스)의 공식 출처는 어디인가.
② 그 데이터를 공개 저장소에 정적 JSON으로 재배포해도 되는가(약관·저작권).
③ 등수별 당첨금액·당첨 게임 수도 같은 출처에서 얻을 수 있는가.
④ 최신 회차는 몇 회이고, 전 회차를 넣을 때 파일 크기는 문제없는가.

## 결론

로또 6/45 당첨번호 및 등수별 당첨금 데이터는 동행복권 공식 "로또 회차별 당첨번호" 페이지의
엑셀다운로드를 기준 출처로 삼는다. 다만 공식 사이트의 공개 재배포 허용 라이선스가 확인되지
않았으므로, **전체 회차 원본 JSON의 공개 재배포는 동행복권 문의(또는 별도 검증) 전까지 보류**한다.
저장소·배포물에 데이터를 어떤 형태로 포함할지는 별도 결정 사항이다(아래 "필요한 검증" 참조).

## 사실 근거 (출처 있음 · 사용자 확인)

- 동행복권 "로또 6/45 회차별 당첨번호" 페이지가 `회차`, `당첨번호`, `보너스`, `순위`,
  `당첨게임수`, `1게임당 당첨금`을 제공하고 **인쇄/엑셀다운로드** 기능이 있다.
  → 당첨번호와 당첨금(③) 모두 공식 출처에서 확보 가능.
- 동행복권 추첨결과 페이지의 회차 선택 목록은 **1회~1231회**를 노출한다(2026-07-06 확인).
  1231회는 2026-07-04 추첨.
- 동행복권 사이트 하단에 "Copyright (c) 2026 복권위원회&동행복권. All rights reserved" 표기.
  별도 오픈데이터 라이선스(재배포 허용 표시)는 확인되지 않음.
- 한국 저작권법상 데이터베이스제작자는 데이터베이스의 전부 또는 상당한 부분을
  복제·배포·전송할 권리를 가진다(개별 사실 데이터와 별개로 DB 자체가 보호될 수 있음).
- 공공데이터포털의 "온라인복권 1등 당첨 판매점 현황 정보"는 이용허락범위 제한 없음으로
  표시되지만, **판매점 현황 데이터라서 당첨번호 재배포 허가의 근거가 되지 않는다.**

## 검증 전 의견·자문 (사실 근거 아님)

- 재배포 방식별 위험 평가(외부 자문 — 법률 자문 아님, 법률 판단은 자문으로 대체 불가):
  - 공식 페이지 링크 제공: 가장 안전
  - 앱 화면에 당첨번호 표시(출처 표기): 가능성 높음
  - 전체 회차 JSON을 공개 저장소에 포함: 약관·DB권 리스크 있음, 사전 문의 없이 비추천
  - 저장소에는 변환 스크립트만 두고 사용자가 직접 생성: 상대적으로 안전
- 파일 크기 추정(대략 계산): 당첨번호만 약 100~150KB, 당첨금 포함 약 500KB~900KB(1,231회 기준)
  → 기술적으로는 전 회차 포함에 부담 없음.
- 데이터 구조 자문: 당첨번호 JSON과 당첨금 JSON은 분리 권장(당첨금은 화면 필수가 아닐 수 있고
  파일 크기·업데이트 검증이 쉬워짐).

## 불확실성

- 동행복권 약관 전문에서 데이터 수집·재배포를 명시적으로 금지/허용하는 조항의 존재 여부는
  확인하지 못했다(저작권 표기만 확인). "허용 확인 안 됨"이지 "금지 확인됨"이 아니다.
- 저작권법 출처 링크(lsiSeq=92099)가 현행 조문인지(개정 반영 여부)는 확인하지 못했다.
- 파일 크기는 추정치다(실측 아님).
- 최신 회차(1231회)는 시점에 따라 바뀐다 — 매주 토요일 추첨.

## 필요한 검증 (전체 JSON 공개 재배포를 하려면)

- 동행복권(또는 복권위원회)에 당첨번호 데이터 재배포 가능 여부를 직접 문의해 답변을 확보하거나,
  약관 전문에서 허용/금지 조항을 확인한다. **검증 전에는 전체 회차 JSON을 공개 저장소·배포물에
  포함하는 결정의 사실 근거로 쓰지 않는다.**

## 사용자 결정 (2026-07-06)

**전체 회차 JSON을 저장소·배포물에 포함한다 — 위험 감수.** 외부 자문의 "사전 문의 없이 비추천"
및 Claude의 보수안+문의 병행 추천과 달리, 사용자가 리스크를 인지하고 전체 포함을 선택했다.
완화 조치로 다음을 적용한다:

- 앱·데이터 파일에 출처 표기: "당첨번호 출처: 동행복권 로또 6/45 회차별 당첨번호"
- 비공식·재배포 허가 아님 명시, 권리자 요청 시 삭제 수용 문구
- 이 결정은 약관·DB권 리스크가 해소된 것이 아니라 **사용자가 감수한 것**이다.
