## 목적

1차 MVP에서 미뤄둔 "로또 전문가 패러디" 기능. 생성된 번호에 로또 "전문가"들이 하는 그럴듯한
분석 멘트를 농담·패러디로 붙여 재미를 더한다. 예측 기능이 아니라 예측 무용론을 웃기게 비트는
재미 요소다.

## 요구사항

1. 뽑기마다 패러디 멘트 풀(15개 이상)에서 랜덤으로 1개를 뽑아 용지("내가 뽑은 번호") 카드
   하단에 보여준다 (2026-07-05 사용자 결정 — 결과 컷 아님).
2. 멘트는 같은 결과가 화면에 있는 동안 바뀌지 않는다(리렌더에 안 흔들림 — 뽑기 시점에 확정).
3. "패러디·과학적 근거 없음" 표시를 멘트와 함께 붙인다(진지한 분석으로 오인 금지).
4. 멘트 어디에도 "당첨 보장"·"확률을 높인다"류 표현이나 '당첨' 단어를 쓰지 않는다
   (기존 사행성 규칙 — 모드명에서 '당첨'을 뺀 결정과 일관).
5. RNG 주입으로 멘트 선택이 결정적으로 테스트 가능해야 한다(기존 도메인 테스트 패턴).

## 완료 조건

- [ ] 뽑기 후 용지 카드에 패러디 멘트 1개와 "패러디·과학적 근거 없음" 표시가 보인다.
- [ ] 멘트 풀 전체에 금지 표현('당첨'·'보장'·'확률을 높')이 없다 — 테스트로 검증.
- [ ] RNG 주입 시 선택이 결정적이고, 어떤 rng 값에서도 유효한 멘트를 돌려준다(경계값 0·1 근처).
- [ ] 기존 테스트 전부 통과 + lint·typecheck·build 통과.

## 범위 밖

- 멘트 공유·복사 기능, 멘트 새로고침 버튼
